This project is the home to various Terminal Reality centered libraries, tools and game remakes

jtrfp

Latest release: 0.1.0

Offers Terminal Reality file format parsers written in Java allowing access to the assets of games like Monster Truck Madness, Fury 3 or Terminal Velocity.

Source/GitHub Downloads Wiki

</dependency>
